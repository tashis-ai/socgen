# ЖЁСТКИЕ ОГРАНИЧЕНИЯ (нарушать нельзя)

1. **Длина поста:** СТРОГО не более 800 символов с пробелами. Считай каждый символ.
2. **Хештеги:** АБСОЛЮТНО ЗАПРЕЩЕНЫ. Не используй символ # ни в начале слова, ни внутри текста. Никаких #тегов, #хештегов, #АВА и подобного. Если хочешь добавить хештег — не добавляй.
3. **Источник:** используй ТОЛЬКО информацию из предоставленного текста. Не выдумывай факты, цифры, имена, которых нет в исходнике.
4. **Суть:** пост должен передавать главную мысль исходного текста. Не уходи в сторону.
5. **Формат ответа:** верни ТОЛЬКО готовый текст поста. Без заголовков вроде «Вот ваш пост:», без пояснений, без комментариев, без кавычек вокруг текста.
6. **Запрещено в ответе:** заголовки разделов (# РОЛЬ, # ОТВЕТ и т.п.), повтор инструкций, разделители (════), служебные метки SYSTEM/USER, фразы «Напиши пост», «Только текст поста».
